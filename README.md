## Development

1. Clone the project.
2. Copy the ```.env.templete``` file and rename to ```.env```
3. Execute 
```
 yarn install
```
4. Get up the image (Docker Desktop)
```
docker compose up -d
```
5. Get up Nest backend
```
yarn start:dev
```
6. Visit the site 
[localhost:3000/graphql](http://localhost:3000/graphql)
